# Web公開および収益化（広告・アフィリエイト）導入ガイド

「結の方程式」を実際に世界へ公開し、収益を上げるための手順をまとめました。

## 1. Webサイトの公開方法 (Deployment)

Next.jsで作られたこのサイトを公開するには、**Vercel（ヴェルセル）** というサービスが最も推奨されます（Next.jsの開発元が運営しており、非常に高速で無料枠も充実しています）。

### 公開のステップ

1. **GitHubへアップロード**:
   - 現在のコードをGitHubのリポジトリにプッシュします。
2. **Vercelと連携**:
   - [Vercel](https://vercel.com/) にサインアップし、GitHubリポジトリを選択して「Import」をクリックします。
3. **デプロイ**:
   - 設定はデフォルトのままで「Deploy」を押すと、数分で `https://yui-no-houteishiki.vercel.app` のようなURLで公開されます。
4. **独自ドメインの設定**:
   - 独自のドメイン（例: `yui-uranai.jp`）を取得し、Vercelの設定画面から紐付けることで、プロフェッショナルな印象になります。

## 2. 広告・アフィリエイトの埋め込み方法

### Google AdSense (ディスプレイ広告) の導入

1. **AdSenseに申し込む**: [Google AdSense](https://www.google.com/adsense/) でサイトを登録し、審査を受けます。
2. **コードの取得**: 審査通過後、「サイトごと」の広告コードを取得します。
3. **実装方法**:
   - `src/app/layout.js` の `head` 部分に自動広告コードを追加します。
   - 個別の場所に貼る場合は、`src/app/result/page.js` の広告プレースホルダー部分に提供された `ins` タグを貼り付けます。

```javascript
// src/app/layout.js への追加例
<head>
  <script
    async
    src="https://pagead2.googlesyndication.com/pagead/js/adsbygoogle.js?client=ca-pub-XXXXXXXXXXXXXXXX"
    crossorigin="anonymous"
  ></script>
</head>
```

### アフィリエイト (専門占いへの誘導) の導入

A8.net、もしもアフィリエイト、バリューコマース等で「電話占い」や「チャット占い」の案件を探します。

1. **アフィリエイトリンクの取得**: 各ASPからバナーまたはテキストリンクを取得します。
2. **実装方法**:
   - `src/app/result/page.js` のアフィリエイトセクションにある「専門鑑定士に詳細を依頼する」ボタンの `href` を、あなたの専用リンクに書き換えます。

```javascript
// src/app/result/page.js の修正例
<a
  href="あなたの専用アフィリエイトURL" // ここを書き換える
  target="_blank"
  className="mystic-button"
>
  専門鑑定士に詳細を依頼する
</a>
```

## 3. 審査を通過するためのヒント

- **プライバシーポリシー**: 必須です。広告の利用やデータの取り扱いについて明記したページを一つ作成してください。
- **免責事項**: 「鑑定結果はエンターテインメントであり、将来を保証するものではありません」といった一文をフッター等に追加することをお勧めします。
- **PR表記（ステマ規制対応）**: 広告やアフィリエイトリンクが含まれるページには、必ず「PR」や「広告」といった表記を目立つ位置に追加してください（実装済みですが、追加で広告を貼る際も同様の対応が必要です）。
- **コンテンツの拡充**: 現在の鑑定の種類をさらに数種類増やすと、Google等の審査に通りやすくなります。
